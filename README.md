# 🎯 Magic Number Game  
A fun mathematical card trick — built with **Bootstrap 5**, **flip animations**, and **sound effects**.

The user thinks of ANY number between **1–100**, the system shows 7 magic cards, and based on YES/NO responses, the game **automatically guesses the number**.

## 🧠 How It Works
The 7 cards are based on **binary mathematics**.  
Each card corresponds to values:  
`1, 2, 4, 8, 16, 32, 64`.

If the user says **YES**, the system adds that value.  
Final number = sum of all YES cards.

Example:  
YES on 1 + 4 + 16 + 32 = **53**

This is your magic number 🎉
